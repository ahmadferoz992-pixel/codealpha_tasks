A basic network packet sniffer built with Python using the Scapy library.
What it does
Captures live network traffic packets, displays source and destination IP addresses, shows protocol type like TCP, UDP, and ICMP, shows port numbers and TCP flags, and prints a preview of the packet payload.
Files
network_sniffer.py is the main sniffer script.
Requirements
Install scapy by running: pip install scapy
How to Run
Run the script using: sudo python3 network_sniffer.py
Note: Root or admin privileges are required for packet capturing.
Tech Used
Python 3 and Scapy.
